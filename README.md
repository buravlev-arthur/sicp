### Решение задач из курса "Structure and Interpretation of Cumputer Programs"
Хранилище разделено по каталогам-главам (Например, "Chapter 1" - задачи из 1 главы книги). Решение задач реализовано на языке [Racket](https://racket-lang.org/), заимствующем каноничный диалект языка [Scheme](https://en.wikipedia.org/wiki/Scheme_(programming_language)), разработанным авторами курса - сотрудниками MIT (Массачусетский технологический институт).
